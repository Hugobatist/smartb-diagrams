  <action>
    Create `static/ws-client.js` -- a plain JavaScript file (no bundling, consistent with existing annotations.js and diagram-editor.js pattern).

    The file exposes a global function `createReconnectingWebSocket(url, onMessage, onStatusChange)` that returns `{ close }`.

    Implementation:
    - Internal state: `ws` (WebSocket instance), `attempt` counter, `timer` (setTimeout ID)
    - Constants: `BASE_DELAY = 500` (500ms), `MAX_DELAY = 16000` (16s cap), `MAX_ATTEMPTS = Infinity`
    - `connect()` function:
      - Guard: if `ws` exists and `readyState < 2` (CONNECTING or OPEN), return
      - Create `new WebSocket(url)`
      - `onopen`: reset `attempt = 0`, call `onStatusChange('connected')`
      - `onmessage`: try `JSON.parse(event.data)`, call `onMessage(parsed)`, catch parse errors to console.error
      - `onclose`: call `onStatusChange('disconnected')`, call `scheduleReconnect()`
      - `onerror`: no-op (onclose fires after onerror)
    - `scheduleReconnect()` function:
      - Calculate `delay = Math.min(BASE_DELAY * Math.pow(2, attempt), MAX_DELAY)`
      - Add jitter: `jitter = delay * (0.5 + Math.random() * 0.5)` (50-100% of delay)
      - Increment `attempt`
      - Call `onStatusChange('reconnecting')`
      - Set `timer = setTimeout(connect, jitter)`
    - `close()` function: `clearTimeout(timer)`, if `ws` call `ws.close()`
    - Call `connect()` immediately
    - Return `{ close }`

    No dependencies. Uses browser-native WebSocket API only.
  </action>

    - No npm imports (pure browser JS)

    1. **Add script tag** for ws-client.js -- add `<script src="ws-client.js"></script>` BEFORE the annotations.js script tag (line ~1318), so it's available when the init block runs.

    2. **Remove polling setInterval** -- delete `setInterval(syncFile, 2000);` (line ~958). Keep the `syncFile()` function itself (it's still useful for initial load and manual sync).

    3. **Remove the file tree polling** -- delete `setInterval(refreshFileList, 5000);` (line ~1297). Tree updates will come via WebSocket.

    4. **Add WebSocket initialization** in the init IIFE (the `(async () => { ... })()` block at line ~1274). After the initial file load, create the WebSocket connection:

       ```javascript
       // WebSocket real-time sync
       const wsProtocol = location.protocol === 'https:' ? 'wss:' : 'ws:';
       const wsUrl = wsProtocol + '//' + location.host + '/ws';

       const wsConnection = createReconnectingWebSocket(wsUrl, (msg) => {
         switch (msg.type) {
           case 'file:changed':
             // Only update if the changed file is the currently viewed file
             if (msg.file === currentFile) {
               const text = msg.content;
               if (text !== lastContent) {
                 // Merge flags if user has local flags
                 let finalText = text;
                 if (window.SmartBAnnotations && SmartBAnnotations.getState().flags.size > 0) {
                   finalText = SmartBAnnotations.mergeIncomingContent(text);
                 } else if (window.SmartBAnnotations) {
                   const incoming = SmartBAnnotations.parseAnnotations(text);
                   SmartBAnnotations.getState().flags = incoming.flags;
                   SmartBAnnotations.getState().statuses = incoming.statuses;
                   SmartBAnnotations.renderPanel();
                   SmartBAnnotations.updateBadge();
                 }
                 lastContent = finalText;
                 editor.value = finalText;
                 render(finalText);
               }
             }
             break;
           case 'file:added':
           case 'file:removed':
           case 'tree:updated':
             // Refresh the file tree
             refreshFileList();
             break;
         }
       }, (status) => {
         // Update connection status indicator
         const dot = document.getElementById('statusDot');
         const text = document.getElementById('statusText');
         switch (status) {
           case 'connected':
             dot.className = 'status-dot';
             text.textContent = 'Connected';
             break;
           case 'disconnected':
             dot.className = 'status-dot paused';
             text.textContent = 'Disconnected';
             break;
           case 'reconnecting':
             dot.className = 'status-dot paused';
             text.textContent = 'Reconnecting...';
             break;
         }
       });
       ```
