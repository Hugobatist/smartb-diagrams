**static/session-player.js** (NEW, ~250 lines):
- IIFE pattern: `var SmartBSessionPlayer = (function() { ... })();`
- Internal state:
  ```
  var state = {
    events: [],
    currentIndex: 0,
    playing: false,
    speed: 1,
    animFrameId: null,
    lastFrameTime: 0,
    diagramStates: [],  // precomputed cumulative state at each event index
    visible: false,
  };
  ```

- **Cumulative State Model:**
  Each diagram state is: `{ nodeIds: Set, edgeIds: Set, statuses: Map, labels: Map }`.
  On session load, precompute `diagramStates[i]` for each event index by applying events sequentially:
  - `node:visited`: add nodeId to nodeIds
  - `node:added`: add nodeId to nodeIds
  - `node:removed`: delete nodeId from nodeIds
  - `edge:added`: add edgeId to edgeIds
  - `edge:traversed`: add edgeId to edgeIds
  - `edge:removed`: delete edgeId from edgeIds
  - `status:changed`: update statuses Map
  - `flag:added` / `flag:removed`: no-op for visual state
  - `risk:set`: update a risk map (optional)
  This precomputation makes seeking O(1) -- just look up `diagramStates[targetIndex]`.

- **computeDiff(prevState, currState):** Returns `{ added: [], removed: [], modified: [] }` by comparing nodeId Sets and label/status Maps. Added = in curr not in prev. Removed = in prev not in curr. Modified = same nodeId but different status or label.

- **applyDiffHighlights(diff):** Clear previous highlights. For added nodes: find element via DiagramDOM, add `diff-added` CSS class. For removed nodes: create temporary ghost rect at last known position (using DiagramDOM.getNodeBBox from previous frame -- store bboxes in state), add `diff-removed-ghost` class. For modified nodes: add `diff-modified` class.

- **clearDiffHighlights():** Remove all `diff-added`, `diff-removed`, `diff-modified` CSS classes from SVG elements. Remove any ghost rect elements.

- **applyFrame(index):** Get `diagramStates[index]`. If index > 0, compute diff from `diagramStates[index - 1]`. Apply diff highlights. Update scrubber position and frame counter text.

- **play():** Set playing=true, record lastFrameTime, start requestAnimationFrame(tick).

- **pause():** Set playing=false, cancel animFrameId.

- **tick(now):** If not playing or at last event, pause and return. Calculate elapsed = (now - lastFrameTime) * speed. Get time delta to next event from timestamps. If elapsed >= timeDelta, advance currentIndex, update lastFrameTime, applyFrame. Request next frame.

- **seekTo(index):** Set currentIndex to clamped value. applyFrame(index). Update scrubber.

- **loadSession(sessionId):** Fetch `GET /api/session/${sessionId}`, parse events, precompute diagramStates, set currentIndex=0, show player bar, applyFrame(0), update scrubber max.

- **handleSessionEvent(sessionId, event):** If a session is being recorded live and player is visible, append event to events, recompute last state, optionally auto-advance if at end.

- **show():** Set visible=true, unhide player DOM element.

- **close():** Pause playback, clear diff highlights, hide player DOM element, set visible=false, reset state.

- **updateScrubber():** Set scrubber input value to currentIndex, update frame counter text `"${currentIndex + 1} / ${events.length}"`.

- **init():** Find DOM elements (#sessionPlayer, #spPlayPause, #spScrubber, #spSpeed, #spTime, #spClose). Wire event listeners:
  - Play/Pause button click toggles play/pause
  - Scrubber input event calls seekTo
  - Speed select change updates state.speed
  - Close button calls close()
  - Keyboard: Space toggles play/pause when player is visible, Left/Right arrow seek by 1 frame

- **Public API:** `{ init, loadSession, handleSessionEvent, show, close, isVisible: function() { return state.visible; } }`
