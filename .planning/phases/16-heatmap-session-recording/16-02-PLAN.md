- Register 4 tools:

1. **start_session** (Tool 8): Takes filePath. Calls `sessionStore.startSession(filePath)`. Returns `{ sessionId, message: "Session started" }`. If no sessionStore (MCP-only mode), return error text.

2. **record_step** (Tool 9): Takes sessionId, nodeId, action, metadata. Builds SessionEvent with `type: 'node:visited'`, `ts: Date.now()`, `payload: { nodeId, action, ...metadata }`. Calls `sessionStore.recordStep(sessionId, event)`. If wsManager available, broadcast `{ type: 'session:event', sessionId, event }` using broadcastAll. Returns `{ recorded: true }`.

3. **end_session** (Tool 10): Takes sessionId. Calls `sessionStore.endSession(sessionId)`. Returns the SessionSummary. If wsManager available, broadcast heatmap:update with aggregated data for the session's diagram file.

4. **set_risk_level** (Tool 11): Takes filePath, nodeId, level, reason. Calls `service.setRisk(filePath, nodeId, level, reason)`. Returns `{ set: true, nodeId, level }`.
