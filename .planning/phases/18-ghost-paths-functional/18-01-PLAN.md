# Plan 18-01: Backend @ghost Annotation Parsing, Serialization, and Persistence

<objective>
Add @ghost as the 5th annotation type in the backend annotation system. Ghost paths persist as `%% @ghost FROM TO "label"` lines in .mmd files, are parsed by parseAllAnnotations, serialized by injectAnnotations, and managed through DiagramService CRUD methods. MCP tools and REST routes switch from in-memory GhostPathStore to file-based persistence via DiagramService.
</objective>
