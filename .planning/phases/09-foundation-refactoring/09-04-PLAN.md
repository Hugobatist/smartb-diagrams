**Refactor `annotations.js`:**

1. Replace all inline SVG ID regex parsing with DiagramDOM calls:
   - The `extractNodeId(element)` helper (which walks up DOM and matches `flowchart-*-\d+`, `subGraph*-*-\d+`, `L-*`) should be replaced with `DiagramDOM.extractNodeId(element)`
   - Any `document.querySelector('#preview svg')` calls should use `DiagramDOM.getSVG()`
   - Node element lookups by ID regex should use `DiagramDOM.findNodeElement(nodeId)`
   - Subgraph lookups should use `DiagramDOM.findSubgraphElement(subgraphId)`

2. Add event bus subscriptions:
   - Subscribe to `diagram:rendered` to re-apply flags to SVG after each render (this currently happens via a direct call from render() in the init block)
   - Emit `flags:changed` when flags are added/removed

3. Keep the hooks pattern working alongside event bus during migration:
   - The `init(hooks)` method must still work — existing callers (app-init.js) still use it
   - Event bus subscriptions are ADDITIONAL, not replacements for hooks (yet)
   - The hooks can be removed in a future cleanup phase

4. IMPORTANT: Do NOT change the public API of SmartBAnnotations. All existing methods (init, toggleFlagMode, togglePanel, getCleanContent, applyFlagsToSVG, getState, parseAnnotations, mergeIncomingContent, renderPanel, updateBadge) must still exist and work.

**Refactor `collapse-ui.js`:**

1. Replace SVG queries with DiagramDOM:
   - Node finding: use `DiagramDOM.findNodeElement(nodeId)` instead of iterating `svg.querySelectorAll('[id]')` with regex match
   - Subgraph finding: use `DiagramDOM.findSubgraphElement(subgraphId)`
   - `getBBox()` calls: can continue using direct `element.getBBox()` on the element returned by DiagramDOM (DiagramDOM returns the element, caller uses getBBox)

2. Add event bus subscription:
   - Subscribe to `diagram:rendered` to re-apply collapse overlays after render

3. Keep the existing `init(callbacks)` pattern working for backward compat.

4. IMPORTANT: Do NOT change the public API of SmartBCollapseUI. All existing methods must still work.
